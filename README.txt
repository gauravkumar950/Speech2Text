The first step is to set up  the ChromeDriver by executing the Chromedriver.py file

Execute the following command in the command prompt to get the directory of the Chromedriver.

              """dir /s /b chromedriver.exe"""


Now put the source file path of the Chrome driver and the destination file where you have put the speech2text project file

Now You are ready to go, Go to the terminal and execute python app.py .Note that you should be  in the current directory off the project.
